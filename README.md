<h1>BookStoreAPI</h1>

## Tecnologias:
<br>

* Ruby
* Sinatra 
* MYSQL
* PostgreSQL


## Projeto:
<br>

<p>
API criada com Sinatra para consultas no banco de dados da livraria e manipulação de dados/tabelas via ActiveRecord. O SGBD utilizado foi o MYSQL(posteriormente houve uma migração para o PostgreSQL).
